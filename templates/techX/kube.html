Kubernetes and KubeFlow

Installation of Kubernetes


sudo apt-get update
sudo apt-get install docker.io
docker ––version
sudo systemctl enable docker
sudo systemctl status docker
sudo systemctl start docker
sudo apt-add-repository "deb http://apt.kubernetes.io/ kubernetes-focal main" 
sudo apt-get install kubeadm kubelet kubectl
sudo apt-mark hold kubeadm kubelet kubectl
kubeadm version

Installation of KubeFlow


export KIND_VERSION=v0.8.1
export YOUR_KIND_DIRECTORY=/home/yui/Documents/prog/kind
mkdir -p /home/yui/Documents/prog/kind
curl -Lo ./kind https://kind.sigs.k8s.io/dl/${KIND_VERSION}/kind-linux-amd64 && \
    chmod +x ./kind && mv ./kind ${YOUR_KIND_DIRECTORY}/kind
export PATH=$PATH:/home/yui/Documents/prog/kind/
./kind
