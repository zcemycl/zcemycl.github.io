Ubuntu 16.04 Virtual Network Computing

To remote control another machine Ubuntu 16.04 in LAN from Ubuntu 18.04, you can use Remmina. Noteworthy to point out, this tutorial is not for the host with Ubuntu 18.04 since it has a bug?

Ubuntu 16.04 VNC server

  1. In Ubuntu 16.04, search vino-preferences under Desktop Sharing.
  2. Tick Allow other users to view your desktop.
  3. Tick Allow other users to control your desktop.
  4. Tick Require the user to enter this password.
  5. Enter a password.
  6. Close.

Ubuntu 18.04 Remmina Remote Desktop Client

  1. Protocol: VNC
  2. Server: 192.168.1.102
  3. Connect
  4. Enter password previously set.
